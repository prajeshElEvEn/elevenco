Markdown is a simple language that is rendered as html. Here's all that you need to know.

<!--truncate-->

## headings

```md
# heading 1
```
```md
## heading 2
```
```md
### heading 3
```
```md
#### heading 4
```
```md
##### heading 5
```
```md
###### heading 6
```
### with id
```md
# heading {h1}
```

## list

```md
- item 1
- item 2
- item 3
```

```md
1. item 1
2. item 2
3. item 3
```

```md
- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
```

```md
First Term
: This is the definition of the first term.

Second Term
: This is one definition of the second term.
```

| ----------- | ----------- |
