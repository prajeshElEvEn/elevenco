Markdown is a simple language that is rendered as html. Here's all that you need to know.

<!--truncate-->

## headings

```md
#

##

###

####

#####

######
```

## list

```md
- item 1
  - item 11
    - item 111
  - item 12
```

```md
1. item 1
2. item 2
3. item 3
```

## styles

```md
**bold**
```

```md

```

```md
~~strike through~~
```
